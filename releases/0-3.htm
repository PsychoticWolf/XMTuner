XM Tuner

XM Tuner allows you to stream your XM Radio Online channels to your UPnP compatible media player (such as the Xbox 360, Playstation 3 and a variety of other set-top wireless media players). Its designed to be used together with a media server such as TVersity.

Release 0.3

December 08, 2009

This release represents a great step forward in the reliability and stability of the application. Numerous crashes from 0.2 have been fixed. A new service option has been added, complete with controls inside the UI for the applicaiton itself. There is also now an update checker to ensure that you have the latest version of the software available to you.

The release is availible from: http://www.pcfire.net/XMTuner/InstallXMTuner0-3.msi
The release notes are available from: http://www.pcfire.net/XMTuner/readme0-3.txt


Changes

  • Can run as a service
  • Updated "What's On" Page
  • Tooltips with channel descriptions
  • Movement of major functionality to tabs
  • Custom addressing by Configuration
  • New configuration format (will require new config for older versions)
  • System-level Configuration (no longer app path)
  • MP3 Transcoding via TVersity (for non-WMA devices)
  • Channel data is cached (less network overhead)
  • "Playing" info updated every 30 seconds
  • More stable than 0.2
  • More pretty than 0.2
  • Automatically checks for new versions
