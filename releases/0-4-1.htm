XM Tuner

XM Tuner allows you to stream your XM Radio Online channels to your UPnP compatible media player (such as the Xbox 360, Playstation 3 and a variety of other set-top wireless media players). Its designed to be used together with a media server such as TVersity.

Release 0.4.1

January 27, 2010

This release is a more robust XMTuner than previous releases. There have been several bug fixes and some requested features added (such as playlists), but the major change for this version is a new player built into the application. It runs on the Windows Media platform (the native platform of XMRO). With the player also comes recently played information as well as a history of what has been played.

The release is availible from: http://www.pcfire.net/XMTuner/InstallXMTuner0-3.msi
The release notes are available from: http://www.pcfire.net/XMTuner/readme0-3.txt


Changes

  • Play XMRO stations directly in the application
  • Updated "Channels" page works with the player
  • "Now Playing" information now available in the application
  • Program Guide added to Channels page
  • History tab tells you last 25 songs played
  • Notification box displays channel information updates
  • PLS, M3U and ASX Playlists available
  • Windows UAC now supported
  • More stable and less crashes than 0.3
